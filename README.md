# Hotel_Management_System
Java hotel management system with console and GUI applications.
## Features

### GUI (Login + Welcome)
- Login screen with username/password
- Welcome screen with buttons for Gym, Spa, Dining, Room Booking (messages only)

### Console (Framework / Business Logic)
- Room reservation with date handling
- Price calculation based on number of nights
- Room search by Room ID
- Logger messages for monitoring events

### Diagrams & Pseudocode
- Class diagrams, flowcharts, and algorithm diagrams
- Pseudocode explaining the system logic


## Technologies Used
- Java (Swing, Scanner, LocalDate)
- Logger for event tracking
- Console and GUI applications

## How to Run
1. Open the project in NetBeans or any Java IDE
2. Run console programs from `src/console` folder
3. Run GUI login screen from `src/gui` folder
